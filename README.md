This bachelor's thesis presents the design and implementation of a privacy-focused fitness application that eliminates reliance on cloud infrastructure. Through an analysis of existing solutions and an assessment of risks associated with sensitive health data, the project establishes clear security requirements as its foundation.
The core innovation lies in replacing cloud-based data transfer and storage with a fully local approach — combining on-device data storage with direct Bluetooth communication between devices. This architecture significantly reduces exposure to data leaks and unauthorized access.
The resulting application delivers a clean graphical interface and flexible tracking of key health metrics, demonstrating that strong privacy protections and a rich, intuitive user experience are not mutually exclusive.
